# PranaLens Wellness Platform

PranaLens Wellness is a full-stack wellness assessment platform built with React, Express, and a JSON mock database.

## Structure
- `frontend/`
- `backend/`
- `database/`
- `assets/`
- `docs/`

## Setup

### Backend
```bash
cd backend
npm install
npm run dev
```

### Frontend
```bash
cd frontend
npm install
npm run dev
```

## URLs
- Frontend: `http://localhost:5173`
- Backend: `http://localhost:5000/api/health`
