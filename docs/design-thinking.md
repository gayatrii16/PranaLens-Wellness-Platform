# Design Thinking Documentation

## 1. Empathy
- Aanya, student: exhausted during exams, wants simple preventive wellness guidance.
- Vivek, product manager: mentally fatigued and wants credible non-invasive advice.
- Neha, HR lead: needs an organization-level Wellness Index to support teams.
- Arjun, dean: wants early student wellbeing signals before issues escalate.
- Kavya, wellness coach: wants AI to reduce reporting effort, not replace her.
- Rohan, founder: wants a premium and professional wellness solution for leaders.
- Samaira, consultant: needs quick insight and trend tracking around travel stress.

## Pain Points
- Generic wellness apps lack personalization.
- Users want clarity on stress, sleep, digestion, and focus together.
- Institutions need measurable summaries, not vague advice.

## Insights
- Non-invasive, premium, and actionable experiences build trust.
- Human interpretation still matters even with AI-assisted analysis.
- Dashboards and appointment flows improve repeat engagement.

## 2. Define
Students, professionals, and organizations need a preventive wellness platform that combines guided assessment, AI-assisted interpretation, and holistic frameworks into practical reports and next steps before imbalance becomes burnout.

## 3. Ideation
- AI-assisted assessment scoring
- Ayurveda-informed recommendation engine
- Acupressure-aligned self-care suggestions
- Dashboard with charts and progress history
- Wellness Index for organizations and campuses
- Knowledge hub for brand credibility

## Mind Map
- Holistic Wellness Platform
  - Assessment
  - Recommendations
  - Appointments
  - Dashboard
  - Knowledge Hub
  - Organization Wellness Index

## 4. Prototype
- Dark glassmorphism UI for a premium startup feel
- Wellness palette using green, blue, and white
- Guided flow: homepage -> trust pages -> auth -> assessment -> dashboard -> booking
- Dashboard visualizes score, categories, and recommendations clearly

## 5. Testing
- User 1 wanted clearer category labels in assessment: added.
- User 2 wanted more actionable tips: split into lifestyle, nutrition, mindfulness.
- User 3 wanted clearer organization services: split services view.
- User 4 wanted a more premium feel: strengthened branding and visuals.
- User 5 wanted campus relevance: added institution messaging throughout.
